<Update label="2026-04-22" description="v5.0.2" tags={["Update", "Compatibility"]}>
  ## CS2 April 2026 patch compatibility

  NinjaHackz v5.0.2 updates the software to work with the CS2 patch released on April 22, 2026. If you are running an older version, you must update to v5.0.2 to continue using the software.

  ### What changed

  - **Game compatibility update** — Refreshed internal data to align with the latest CS2 update. The software now works correctly with the April 22, 2026 patch.
  - **Improved loader reliability** — The loader now automatically detects required game components at runtime instead of relying on fixed values. This reduces the chance of errors during startup, especially after future game updates.
  - **Better loading status messages** — The loading screen now shows detailed progress information while the software initializes. You can see exactly which step the loader is on and diagnose issues more easily if something goes wrong.

  ### What you need to do

  1. Log in to your [account dashboard](https://ninjahackz.com) and download the latest version.
  2. Replace your existing installation with the new download.
  3. Launch the software as usual — your existing license key continues to work.

  <Warning>
    Older versions of NinjaHackz are not compatible with the April 22, 2026 CS2 update. You must update to v5.0.2 to avoid errors.
  </Warning>
</Update>
